# 🛒 Ordering System App

A **cross-platform ordering system** built using **Python**, **Kivy**, and **KivyMD**, designed to help users browse products, manage a cart, and place orders through a clean and intuitive interface. This project is beginner-friendly and demonstrates core app development concepts such as **UI design**, **data handling**, and **order management**.

---

## ✨ Features

* Cross-platform support (Android, iOS, Windows, Linux, macOS).
* Browse menu or product categories easily.
* Add items to cart and adjust quantities.
* View order summary before checkout.
* Simple and customizable design for different themes (restaurant, café, shop, etc.).

---

## 🛠 Technologies Used

* **Python** – Programming language
* **Kivy** – Cross-platform GUI framework
* **KivyMD** – Material Design for Kivy
* (Optional) **SQLite** or **JSON** for local data storage if applicable

---

## 🔨 Process of Building It

1. **Planning & Wireframing:** Designed app flow and screens (menu, cart, checkout).
2. **Setup Environment:** Installed Python, Kivy, KivyMD, and tested on desktop.
3. **UI Development:** Built screens using KivyMD widgets and layouts.
4. **Functionality Implementation:** Added cart management, order summaries, and navigation logic.
5. **Testing & Debugging:** Tested cross-platform behavior and corrected layout issues.
6. **Customization:** Added theming options for different business types.

## ▶️ Running the Project

1. Clone the repository:

```bash
git clone https://github.com/astigPree/FoodRestaurant.git
```

2. Navigate to the project folder:

```bash
cd FoodRestaurant
```

3. Install dependencies:

```bash
pip install kivy kivymd
```

4. Run the app:

```bash
python main.py
```
